# Product Mission

## Pitch
Vegas Trivia is an interactive trivia game that helps Red Hat field sales teams and technical advisors engage Java development teams in conversations about modernization by providing a compelling, hands-on demonstration of Quarkus capabilities during the 2026 sales kickoff.

## Users

### Primary Customers
- **Field Sales Teams**: Sales professionals who need engaging demonstration tools to open conversations with Java development teams about application modernization and cloud-native development.
- **Technical Advisors**: Technical experts who need to articulate migration strategies and demonstrate Quarkus technical capabilities to prospective customers.

### User Personas
**Field Sales Representative** (28-45 years)
- **Role:** Enterprise sales professional targeting Java development organizations
- **Context:** Attending Red Hat 2026 sales kickoff event, seeking tools to demonstrate value propositions
- **Pain Points:** Difficulty engaging technical audiences with traditional sales presentations; need concrete examples of Quarkus benefits
- **Goals:** Create memorable experiences that lead to follow-up conversations; demonstrate Quarkus simplicity and modern capabilities

**Technical Advisor** (30-50 years)
- **Role:** Technical consultant supporting sales engagements
- **Context:** Working with prospective customers evaluating Java modernization options
- **Pain Points:** Need quick, impactful demonstrations of Quarkus features; limited time to show technical advantages
- **Goals:** Clearly communicate migration strategies; showcase modern development practices with Quarkus

## The Problem

### Engaging Technical Audiences in Sales Contexts
Sales teams struggle to engage Java development teams with traditional presentation methods. Technical audiences need hands-on, interactive experiences to understand the value of modernization initiatives. Static presentations fail to demonstrate the simplicity and power of modern Java frameworks like Quarkus.

**Our Solution:** Vegas Trivia provides an interactive, competitive experience that naturally demonstrates Quarkus capabilities while entertaining participants. The game itself becomes a conversation starter about modern Java development, full-stack applications, and cloud-native architecture.

## Differentiators

### Demonstration Through Experience
Unlike traditional slide presentations or static demos, Vegas Trivia allows participants to experience a working Quarkus application firsthand. The game demonstrates real-world capabilities including session management, data persistence, real-time updates, and modern web development practices.

### Built-In Messaging Platform
The application incorporates randomized informational messages alongside trivia questions, naturally exposing users to key Quarkus value propositions without disrupting the game experience. This creates multiple touchpoints for technical messaging throughout the gameplay.

### Event-Optimized Design
Designed specifically for the sales kickoff context with lightweight architecture (in-memory storage), simple username-based access (no complex authentication), and instant deployment capabilities. This enables quick setup and immediate engagement without infrastructure complexity.

## Key Features

### Core Features
- **Simple Username Login:** Users enter a username to join the game, with credentials stored in browser memory for seamless gameplay without session management overhead.
- **Interactive Trivia Questions:** Users answer multiple-choice questions with automatic progression to the next question after submission, creating continuous engagement.
- **Real-Time Leaderboard:** Displays player rankings based on correct answers, fostering friendly competition and sustained participation.

### Engagement Features
- **Randomized Messaging:** Three rotating informational messages about the Quarkus application display alongside questions, naturally integrating product education into gameplay.
- **Instant Feedback:** Immediate answer submission and scoring keeps users engaged and provides instant gratification.

### Technical Features
- **Full-Stack Quarkus Architecture:** Demonstrates Quarkus web capabilities by serving both backend API and frontend SPA from a single application.
- **In-Memory Data Management:** Uses HashMap-based storage for questions, answers, and leaderboard, showcasing simplicity and rapid development.
